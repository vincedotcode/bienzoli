# Case Study — DS Nails

## Client
- **Name:** DS Nails
- **Industry:** Beauty / Nail Salon
- **Location:** Quatre Bornes, Mauritius
- **URL:** dsnail.vercel.app

## Challenge
A home-based salon was growing through Instagram but lacked a central booking and pricing destination to convert profile visitors into confirmed appointments.

## Solution
Built a salon website with service clarity and booking intent built in:
- Service catalogue with pricing and duration
- Visual gallery for style proof
- Testimonial section
- Booking/auth capability
- Instagram integration

## Key Features
- Service cards with price and duration
- Gallery section
- Client testimonial blocks
- Authentication-enabled booking flow
- Instagram follower/social proof integration

## Result
DS Nails now has a professional conversion channel that turns social traffic into structured bookings and reduces friction for first-time clients.

## Tech Stack
- Next.js
- Tailwind CSS
- Vercel
- Authentication system
