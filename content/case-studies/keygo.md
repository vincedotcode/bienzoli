# Case Study — KeyGo

## Client
- **Name:** KeyGo
- **Industry:** Car Rental
- **Location:** Mauritius
- **URL:** keygomu.vercel.app

## Challenge
A new car rental brand needed to look premium and trustworthy while serving both tourists and locals on mobile-first booking journeys.

## Solution
Built a modern rental platform emphasizing clarity and local conversion readiness:
- Fleet display with clear vehicle specs
- Booking flow optimized for fast enquiries
- Local payment method familiarity signals
- Testimonial-led trust reinforcement

## Key Features
- Fleet gallery with specs and visual hierarchy
- Mauritian payment method display (MCB Juice, my.t money, Emtel Cash, MauCAS QR)
- Airport pickup messaging
- WhatsApp booking integration
- Mobile-optimized responsive flow for tourist usage

## Result
KeyGo gained a polished digital presence that differentiates from generic rental pages and aligns with local payment behavior, improving user trust and enquiry readiness.

## Tech Stack
- Next.js
- Tailwind CSS
- Vercel
