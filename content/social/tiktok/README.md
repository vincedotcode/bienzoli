# TikTok Content Guidelines

## Format
- Aspect ratio: 9:16 vertical
- Duration: 15-60 seconds
- Hook: first 3 seconds (Kreol or English)

## Content Types
- 48-hour build timelapses
- Speed test comparisons
- Before/after website transformations
- "This business needs a website" roasts (respectful)
- Client testimonial clips

## Frequency Target
- 3 posts per week

## Structure
1. Hook
2. Visual proof
3. Short insight
4. CTA

## CTA Standard
End with: `bienzoli.com` or `DM moi`.

## Audio Rule
Use trending sounds with proper rights usage. Avoid copyrighted material without permission.
