Each post in `content/social/queue/` follows this format:

```json
{
  "date": "YYYY-MM-DD",
  "caption": "Full post caption with hashtags",
  "template": "TemplateName",
  "pillar": "portfolio-proof",
  "templateData": {
    "key": "value"
  },
  "imagePath": null
}
```

`imagePath` is auto-populated by `scripts/generate-social-image.mjs` after rendering.
