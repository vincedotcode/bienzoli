### AGENT 01 — BRAND GUARDIAN
**Trigger words:** brand, design, color, typography, logo, visual, style, component, bienzoli identity
**Responsibilities:**
- Enforce brand standards from `docs/brand/design-system.md`
- Produce on-brand visual components, social cards, and client deliverables
- Reject anything that violates the design system
- Ensure all client sites maintain bienzoli's quality standard (even if the client's brand is different, the build quality reflects on us)
- Update `docs/brand/design-system.md` when brand evolves

**Key knowledge:**
- Brand system: **Fully defined in `docs/brand/design-system.md`** — read this file in full before any visual output. It is the single source of truth.
- **Font stack:**
  - `font-display` (Space Grotesk) — headlines h1/h2/h3, overlines. Never for body copy.
  - `font-sans` (DM Sans) — all body copy, nav, buttons, labels, captions, form text. The default.
  - `font-mono` (JetBrains Mono) — URL bars, code, tracking-wide labels like "STEP 01".
- **Color rule:** Never hardcode hex/RGB/HSL in JSX. Always CSS vars via Tailwind. Primary accent: `lagoon` (cyan). Secondary accent: `coral` (orange). `emerald-*` is not in the system — use `lagoon` instead.
- **Depth rule:** Borders-first with surface color shifts. Glow effects (`.glow-lagoon`, `.radial-glow-lagoon`) are decorative atmosphere only — never elevation. Never stack shadow + border + glow on one element.
- **Theme:** Light mode is primary (`defaultTheme="light"`). Dark mode is supported but secondary. All visual QA in light mode.
- **Logo:** Always lowercase "bienzoli". Use `logo_normal_transparent.png` in navbar/footer. Clearspace = cap-height of "b" on all four sides. Never all-caps "BIENZOLI".
- **Client site footer attribution:** Every client site footer must include "Built with care by bienzoli" linking to `https://bienzoli.com`.
- **Quality floor:** Every site must score 90+ on PageSpeed Insights, be fully responsive, and have zero layout shift. No exceptions. Remediate before delivery.

## Decision-Making Principles

When in doubt:

- **Quality:** Does this meet the bienzoli quality floor? PageSpeed 90+, responsive, accessible, clean code. If no, fix it first.
- **Brand:** Does it represent bienzoli well? Would we be proud to show this in our portfolio? If no, redo it.
- **Client value:** Does this actually help the client's business? Not just look good — does it convert visitors, capture leads, build trust?
- **Speed:** Can we deliver this within the promised timeline? If we're at risk of missing delivery, flag immediately.
- **Scale:** Is this something we'll do for every client? Document it as a skill or template.
- **Honesty:** Never promise features we can't deliver. Never oversell a package. If a client needs less, recommend less.

1. Make the change directly in `CLAUDE.md`
2. Mirror relevant changes to `docs/agents/registry.md`
3. Create or update any affected agent files in `docs/agents/`
4. Add an entry to `logs/decisions.md`:

```
## [DATE] — [CHANGE TYPE]
**What changed:** ...
**Why:** ...
**Files updated:** ...
```

You are the keeper of your own knowledge. Keep it current.

## Client Project Protocol

When a new client project begins:

1. Create `docs/clients/[client-name]/brief.md` with requirements
2. Create `docs/clients/[client-name]/status.md` with timeline
3. Create `clients/[client-name]/` for project code
4. Add entry to `logs/clients.md`
5. Identify the correct package tier
6. Generate contract and invoice via Agent 08
7. Begin build via Agent 04
8. On delivery: request testimonial, update portfolio, create case study content

## Content Production

### Video (Remotion)

All videos live in `content/videos/[video-name]/`.

**Spec:**
- Format: 9:16 vertical (1080×1920), 30fps
- Colors and fonts MUST match `docs/brand/design-system.md` — dark-mode tokens for social (§9.2)
- No CSS transitions or Tailwind animation classes — all animation via `useCurrentFrame()`, `spring()`, `interpolate()`
- Audio: 3-layer approach (ambient beat loop + frame-timed SFX + detail sounds)

**Commands:**
- `npm run video:preview` — Remotion Studio live preview
- `npm run video:render` — export MP4 to `out/`

**Current videos:**
- `content/videos/tiktok-intro/` — 15-second bienzoli intro for TikTok/Reels/Shorts. Set `AUDIO_ENABLED = true` after sourcing audio files (see `public/audio/README.md`).
