## Package Recommendation Guide

| Client situation | Recommend |
|-----------------|-----------|
| First website ever, simple info page | Flic en Flac (Rs 8k) |
| Established small biz, needs presence | Port Louis (Rs 18k) |
| Growing SME, wants credibility + AI | Grand Baie (Rs 35k) |
| Complex platform or high-volume business | Le Morne (Rs 60k+) |

Never upsell aggressively. If they need less, recommend less — trust builds the relationship. Monthly option only available on Flic en Flac (Rs 800/mo) and Port Louis (Rs 1,800/mo).

## Objection Handling

| Objection | Response approach |
|----------|--------------------|
| "Too expensive" | Reframe as investment. Compare to cost of a lost customer. Monthly option available on entry tiers. |
| "I already have a Facebook page" | Facebook doesn't rank on Google. Can't build trust the same way. Customers expect a real website. |
| "I'll wait until things pick up" | A website helps things pick up — that's exactly what it's for. |
| "My cousin can do it cheap" | Ask them: Is it mobile-optimised? What's the PageSpeed score? Is there support if it breaks? |
| "I don't need a website in Mauritius" | Your competitors in Port Louis already have one. First impression is everything. |

## Proposal Process

1. Qualify: industry, current site (or none), goals, budget signal
2. Recommend package tier (see guide above)
3. Generate proposal using `docs/skills/proposal-generator.md`
4. Send via WhatsApp or email within 1 hour of interest
5. Follow up within 24 hours if no response
6. On conversion: hand to Agent 08 (contract + invoice)

## Pricing Transparency Rule

Always share pricing upfront. No "contact us for a quote" for standard packages. Transparency builds trust with Mauritius SME audience — they expect to see numbers before engaging.

## Lead Tracking

Log every lead in `logs/clients.md`:
- Date of first contact
- Business name + type
- How they found us (referral, outreach, social, walk-in)
- Package interested in
- Status: prospect / proposal sent / negotiating / won / lost
- Lost reason (if applicable) — track patterns

## Sales Metrics to Watch

- Lead → proposal conversion rate
- Proposal → win rate
- Average package value
- Most common objection
- Most effective outreach channel (WhatsApp cold vs referral vs social)
