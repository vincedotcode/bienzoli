# Content Performance Log — bienzoli Facebook Page

Auto-maintained by `scripts/fetch-engagement.mjs`.
Updated: each time a post is made and when engagement is fetched.

## How to read this log

- **Likes / Reach / Comments / Shares** are fetched 24h after posting
- **Reach** = unique accounts who saw the post
- **Impressions** = total times the post was displayed (can be > reach if users saw it multiple times)
- **Engagement rate** = (Likes + Comments + Shares) / Reach × 100
- Run `npm run engagement:fetch` to update all rows with real data from the Graph API
