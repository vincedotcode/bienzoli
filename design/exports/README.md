# Export Guidelines

## Purpose
Standardize exported assets for social, proposals, and client deliverables.

## Output Buckets
- `social-cards/[client-name]/`
- `og-images/`
- `proposal-assets/`
- `client-deliverables/`

## Naming
`[client]-[asset]-[ratio]-v[version].[ext]`

## Validation
- Correct aspect ratio
- Legible typography on mobile
- Brand compliance check complete
- Compression optimized for platform
