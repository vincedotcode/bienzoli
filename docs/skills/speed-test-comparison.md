# Skill: Speed Test Comparison

## Purpose
Create objective performance comparisons to support sales and content.

## Process
1. IDENTIFY — Select the client's current site or competitor site.
2. TEST — Run PageSpeed Insights for mobile and desktop.
3. CAPTURE — Save score screenshots and key diagnostics.
4. COMPARE — Run same test for a bienzoli-built site.
5. FORMAT — Build side-by-side comparison visual.
6. USE — Share in outreach or publish as proof content.

## Reporting Standard
Include:
- URL tested
- Test date
- Mobile score
- Desktop score
- Top 3 performance blockers observed
