# Skill: AI Chatbot Deploy

## Purpose
Deploy multilingual AI chatbot support safely on client projects.

## Process
1. COLLECT — Gather client FAQ, services, pricing, hours, location, and common objections.
2. TRAIN — Draft system prompt with business scope, tone, and escalation boundaries.
3. BUILD — Implement floating chat widget and panel UI.
4. CONFIGURE — Build API route for OpenAI or Anthropic completion calls.
5. LANGUAGES — Test responses in English, French, and Mauritian Kreol.
6. DEPLOY — Release to production and confirm route/security behavior.
7. MONITOR — Review first-week conversation logs and failure cases.
8. REFINE — Update prompts/knowledge base from real usage patterns.

## Guardrails
- Never invent prices/services outside client brief.
- Route unknown/high-risk requests to human contact fallback.
- Keep responses concise and action-oriented.
