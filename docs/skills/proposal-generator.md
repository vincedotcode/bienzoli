# Skill: Proposal Generator

## Purpose
Produce a clear, scope-safe proposal tailored to client goals.

## Process
1. IDENTIFY — Select the correct package tier based on needs and timeline.
2. CUSTOMIZE — Adapt package description to the client's business context.
3. PRICE — Add approved add-ons (example: copywriting Rs 3,000).
4. TIMELINE — Set realistic dates based on current workload and package SLA.
5. FORMAT — Generate as PDF or structured WhatsApp proposal.
6. SEND — Deliver via client's preferred channel.
7. FOLLOW-UP — Send follow-up message after 3 days if no response.

## Non-Negotiables
- Do not oversell features outside package.
- Do not discount by default; adjust scope if needed.
