# Skill: Portfolio Case Study

## Purpose
Turn a delivered project into reusable proof for sales and marketing.

## Process
1. SCREENSHOT — Capture full-page desktop and mobile screenshots of live site.
2. METRICS — Run PageSpeed test and record mobile/desktop scores.
3. WRITE — Document client, challenge, solution, features, and result.
4. SOCIAL — Trigger social card generation workflow (Agent 10).
5. PUBLISH — Add/update portfolio section on bienzoli.com.
6. ARCHIVE — Save case study in `content/case-studies/[name].md` and sync to Drive.

## Output Checklist
- One complete case study file
- Screenshot set
- Metrics snapshot
- Social asset pack request filed
