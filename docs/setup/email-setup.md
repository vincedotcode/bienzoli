# Professional Email Setup — bienzoli

## Purpose
Set up consistent, professional client communication with `@bienzoli.com` addresses.

## Required Mailboxes
- `hello@bienzoli.com` (public contact)
- `vince@bienzoli.com` (founder mailbox)

## Recommended Providers
- Google Workspace (primary recommendation)
- Zoho Mail (cost-sensitive fallback)

## Setup Checklist
1. Confirm domain ownership for `bienzoli.com`.
2. Create mailbox users.
3. Configure MX records.
4. Configure SPF, DKIM, and DMARC.
5. Test inbound/outbound deliverability.
6. Set signature format and response templates.
7. Update website contact details and docs.
